Commands:

Setup and management
  zmail setup [--email <e>] [--password <p>] [--openai-key <k>] [--no-validate]
  zmail wizard [--no-validate] [--clean] [--yes]
  zmail refresh [--since <spec>] [--foreground]  # initial history / backfill (long-running; --foreground waits; else background + log path)
  zmail rebuild-index

Monitor & manage inbox
  zmail refresh [--force] [--text]               # day-to-day: fetch new mail since last sync (forward sync)
  zmail ask "<question>" [--verbose]
  zmail inbox [<window>] [--since YYYY-MM-DD] [--thorough] [--text]
  zmail rules validate|reset-defaults --yes|list|show <id>|add|edit <id>|remove <id>|move <id>|feedback  (see zmail rules --help; ~/.zmail/rules.json)
  zmail archive <message_id> … [--undo]

Search & read
  zmail search <query> [flags]
  zmail who [query] [flags]
  zmail read <id> [--raw] [--json] [--text]
  zmail thread <id> [--json] [--text]
  zmail attachment list <message_id>
  zmail attachment read <message_id> <index>|<filename>

Sending mail
  zmail send [--to ... --subject ...] [<draft-id>]
  zmail draft list|view|new|reply|forward|edit|rewrite  (see zmail draft --help)

Other
  zmail status [--json]
  zmail stats [--json]
  zmail mcp

Upgrade / reinstall (prebuilt binary):
  curl -fsSL https://raw.githubusercontent.com/cirne/zmail/main/install.sh | bash
  curl -fsSL https://raw.githubusercontent.com/cirne/zmail/main/install.sh | INSTALL_PREFIX=~/bin bash
  curl -fsSL https://raw.githubusercontent.com/cirne/zmail/main/install.sh | bash -s -- --nightly
  If you installed via Homebrew, npm, or cargo, upgrade with that tool instead.

Run zmail <command> --help for command-specific options.
