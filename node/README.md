# Node + TypeScript (`@cirne/zmail`)

Reference implementation and **published npm package**. All `npm` commands run **from this directory** (after `nvm use` from the repository root).

```bash
cd node
npm install
npm run zmail -- --help
npm test
```

See **[AGENTS.md](../AGENTS.md)** for the full command list, eval suite, and `npm run install-cli` (global install + Claude skill).
